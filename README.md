# 📡 Dakar 4G KPI Prediction — VS Code Project

Regression-based prediction of 4 network KPIs using Dakar 4G RAN data.

## Project Structure

```
4G_KPI_VSCode/
├── data/
│   ├── KPI_Dataset_Dakar_4G.csv        ← Place here
│   ├── 4g_Senegal_RAN_Cellfile.xlsx    ← Place here
│   ├── clean_dataset.csv               ← Generated by Phase 1
│   └── column_config.json              ← Generated by Phase 1
├── models/                             ← Saved models (Phase 3)
├── outputs/                            ← Plots & metrics
├── app/
│   └── streamlit_app.py                ← Interactive dashboard
├── src/
│   ├── config.py                       ← Central configuration
│   ├── phase1_data_exploration.py
│   ├── phase2_visualisation.py
│   ├── phase3_model_training.py
│   └── phase4_deployment.py
├── requirements.txt
└── README.md
```

## Prediction Targets

| Target | Description |
|---|---|
| `Total_DataVol_GB` | Total data volume per cell per day (GB) |
| `LTE_Avg_RRC_CU` | Average connected users |
| `LTE_Cell_THP_DL_Mbps` | Downlink cell throughput (Mbps) |
| `LTE_PrbUtil_DL_pct` | Downlink PRB utilisation (%) |

## Setup

```bash
# 1. Create virtual environment
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Place data files in data/ folder
```

## Run the Pipeline

```bash
# Phase 1 — Data exploration & feature engineering
python src/phase1_data_exploration.py

# Phase 2 — Visualisation & model selection
python src/phase2_visualisation.py

# Phase 3 — Model training, benchmarking & evaluation
python src/phase3_model_training.py

# Phase 4 — Deployment & monitoring
python src/phase4_deployment.py
```

## Launch Streamlit Dashboard

```bash
streamlit run app/streamlit_app.py
```

Open your browser at: **http://localhost:8501**

### Dashboard Pages

| Page | Content |
|---|---|
| 🏠 Overview | Project summary, dataset stats, final performance table |
| 📊 Model Performance | Actual vs Predicted, residuals, feature importance (per target) |
| 🔮 Single Prediction | Configure cell parameters and get instant KPI predictions |
| 📈 Monitoring Dashboard | Rolling RMSE with alert thresholds, error percentile table |
| 🔍 Benchmark Comparison | All 8 models ranked by CV RMSE per target |

## Models Benchmarked

Linear Regression · Ridge · Lasso · Decision Tree · Random Forest · Gradient Boosting · XGBoost · LightGBM
